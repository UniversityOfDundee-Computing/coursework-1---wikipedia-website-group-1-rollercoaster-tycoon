Game Reception Page


As the user opens up the game they are greeted by an animated backgound and the screams and cheers of the crowd on the rides.

Rolacoaster tycoon

There are 4 buttons for the user to interact with:

Opening your game
When the button is clicked the game opens the last saved version of the game which will have been done by the user the last time that the game was played.

Saving
To make sure that you can continue the game later you have to save your progress by clicking the button with the blue folder.

New Comers A new player may not know how to play the game o they need to learn and use play the tutorial which will explain to the user how the game works.

When the user has saved the game and is finished playing the game they can leave to their desktop.
